Graphraum is deliberately graph-shaped. It accepts nodes, edges, and positions; renders them with Three.js; and keeps application state outside the engine.

## Current boundary

Graphraum owns the WebGL scene, camera, controls, picking, selection colors, resizing, and resource disposal. Your application owns graph meaning, layout policy, labels, accessibility, and product interactions.

That boundary keeps the engine small enough to reason about and makes performance regressions visible instead of hiding them behind a broad framework API.
